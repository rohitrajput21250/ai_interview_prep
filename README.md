# AI Interview & Learning Ecosystem 🚀

An intelligent AI-powered platform designed to help students and developers prepare for technical interviews through personalized learning roadmaps, AI mock interviews, coding assessments, resume analysis, and real-time performance tracking.

This project combines modern AI capabilities with scalable full-stack architecture to create a production-ready interview preparation ecosystem.

---

## ✨ Features

### 🤖 AI Resume Analyzer
- Upload resumes for AI-powered analysis
- ATS score evaluation
- Skill extraction
- Weak area detection
- Resume improvement suggestions
- AI-generated project description optimization

---

### 🧠 Personalized Learning Roadmap
- Dynamic interview preparation roadmap
- AI adapts roadmap based on:
  - weak topics
  - coding performance
  - interview scores
  - target role
- Covers:
  - DSA
  - System Design
  - Frontend/Backend concepts
  - HR interview preparation

---

### 💻 Coding Assessment Platform
- Monaco code editor integration
- Multi-language coding support
- Real-time code execution
- Custom test cases
- AI hints and optimized solutions
- Coding analytics dashboard
- Submission history
- Leaderboards

---

### 🎤 AI Mock Interview System
- Technical interview simulation
- HR interview mode
- AI-generated follow-up questions
- Adaptive difficulty system
- Voice interview support
- Confidence and communication analysis
- Detailed interview feedback reports

---

### 📊 Analytics Dashboard
Track:
- coding consistency
- interview readiness
- topic mastery
- weak areas
- learning progress
- streaks and activity

Includes:
- radar charts
- heatmaps
- progress graphs
- performance trends

---

### ⚡ Real-Time Features
- Live coding sessions
- Real-time interview rooms
- Collaborative practice
- Socket.IO integration

### 🧠 AI Memory System
The platform remembers:
- previous interview mistakes

Powered by:

# 🧪 Installation

## Clone Repository

```bash
git clone https://github.com/rohitrajput21250/ai_interview_prep.git
```

---

## Install Frontend

```bash
cd client
npm install
npm run dev
```

---

## Install Backend

```bash
cd server
npm install
npm run dev
```

---

# 🔐 Environment Variables

Create a `.env` file in the server directory.

```env
PORT=5000

MONGO_URI=your_database_url

JWT_SECRET=your_secret_key

OPENAI_API_KEY=your_api_key

REDIS_URL=your_redis_url
```
